# Car-eye-streamingGBS-View
GB28181 管理平台代码，采用VUE框架，实现对设备监控，管理等功能

# 功能一览

##  设备管理

平台管理国标设备，可以手工或者excel导入设备数据。

##  设备监控

平台可以实时监控设备视频，支持对讲和报警查询，支持发送命令到设备
控制云台，调整参数等
